# Portfolio

**VaultGuard**: A Zero-Knowledge Password Manager using AES-GCM 256-bit encryption with features like breach detection and a dedicated panic mode.

**Disk Scheduling Visualizer**: An interactive tool for calculating and visualizing Total Head Movement using OS algorithms like FCFS, SSTF, and SCAN.

**SmartCampusPortal**: A comprehensive ASP.NET web portal for managing student registrations, faculty roles, and grading systems via Linq to SQL.
