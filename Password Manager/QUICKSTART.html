üéÆ Usage Tips

Password Generator

  • Adjust length slider for longer passwords (16+ recommended)
  • Include all character types for maximum strength
  • Add personal words to make passwords memorable yet secure
  • Copy generated passwords directly from the generator

Security Audit

  • Click "Check All for Breaches" to verify passwords against known breaches
  • Fix weak passwords immediately - they're your biggest vulnerability
  • Never reuse passwords across different accounts
  • Update passwords that are flagged as old

Backup & Restore

  • Export backups before major changes
  • Backups are encrypted with your master password
  • You'll need your master password to restore from backup
  • Store backups on USB drives, cloud storage, or external drives

  • Encryption: AES-GCM 256-bit
  • Key Derivation: PBKDF2 with 600,000 iterations
  • Integrity: HMAC-SHA256 tamper detection

üÜò Troubleshooting

Can't unlock vault?

  • Verify you're entering the correct master password
  • Check for CAPS LOCK
  • Try clearing browser cache (you may lose data - restore from backup)

Data not saving?

  • Ensure browser allows IndexedDB storage
  • Check browser's storage settings
  • Disable "Clear data on exit" settings

Performance issues?

  • Large vaults (500+ entries) may slow down slightly
  • Security audit with breach checking takes time (rate limited API)
  • Try using a modern browser with hardware acceleration
